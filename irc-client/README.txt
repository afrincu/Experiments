client_irc.php creates an IRC robot, a client that connects to a certain IRC channel and sends specific messages.

By using a chosen nickname, it joins a certain IRC channel. 
After connection, it sends a private message 'ce faci?'

If one of the message son the channel is 'Bonjour' it replies with 'bonjour'.
If one of the message son the channel is 'time' it replies with the current time, in the format hour:minutes:seconds.
